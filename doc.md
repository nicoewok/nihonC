# Changes
This is a list of all syntactical and keyword changes.

## End of line
```;``` has been replaced by the Japanese dot ```。```.

## Keywords

### Type names (WIP)
I changed the most common type names to their Katakana counterpart:

| Katakana | C type |
|----------|--------|
| チャル    | char   |
| イント    | int    |
| ロング    | long   |
| フロート  | float  |
| ダブル    | double |
